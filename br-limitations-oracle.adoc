= Limitazioni note con NetApp Backup and Recovery per carichi di lavoro Oracle
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Le limitazioni note identificano piattaforme, dispositivi o funzioni che non sono supportati da questa versione del prodotto o che non interagiscono correttamente con esso. Esamina attentamente queste limitazioni.

Le seguenti azioni non sono supportate nella versione di anteprima privata dei carichi di lavoro di Oracle Database in NetApp Backup and Recovery:

* Backup offline
* Clone
* Configurazione ASM
* Protezione dei database archiviati su SAN
